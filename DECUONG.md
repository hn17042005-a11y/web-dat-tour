ĐỀ CƯƠNG CHỨC NĂNG
 Đề tài Xây dựng Website Đặt Tour Du Lịch Online
1.  Mục tiêu đề tài

Xây dựng một website cho phép người dùng tìm kiếm, đặt tour du lịch trực tuyến một cách nhanh chóng, tiện lợi. Hệ thống hỗ trợ quản lý tour, người dùng, đơn hàng và thanh toán online.

2.  Đối tượng sử dụng
Admin (Quản trị viên)
User (Khách hàng)
3.  Chức năng hệ thống
 3.1. Chức năng dành cho Admin
3.1.1. Quản lý người dùng
Thêm  sửa  xóa người dùng
Phân quyền (Admin  User)
Khóa  mở khóa tài khoản
3.1.2. Quản lý danh mục tour
Thêm  sửa  xóa danh mục (VD Tour trong nước, Tour quốc tế...)
Hiển thị danh sách danh mục
3.1.3. Quản lý tour (sản phẩm)
Thêm  sửa  xóa tour
Upload hình ảnh tour
Cập nhật giá, lịch trình, số lượng chỗ
Hiển thị danh sách tour
3.1.4. Quản lý tin tức
Thêm  sửa  xóa bài viết
Quản lý nội dung blog du lịch
3.1.5. Quản lý đơn hàng
Xem danh sách đơn hàng
Cập nhật trạng thái đơn (chờ xác nhận, đã thanh toán, hoàn thành, hủy)
3.1.6. Thống kê  báo cáo
Thống kê doanh thu theo thời gian
Thống kê số lượng đơn hàng
Thống kê tour được đặt nhiều
 3.2. Chức năng dành cho User
3.2.1. Trang chủ
Hiển thị tour nổi bật
Hiển thị danh mục tour
Banner quảng cáo
3.2.2. Xem danh sách tour
Xem tất cả tour
Lọc theo danh mục  giá  địa điểm
Tìm kiếm tour
3.2.3. Chi tiết tour
Xem thông tin chi tiết (giá, lịch trình, hình ảnh...)
Đặt tour
3.2.4. Giỏ hàng
Thêm  xóa tour
Cập nhật số lượng người
Xem tổng tiền
3.2.5. Thanh toán online
Thanh toán qua
VNPay
MoMo
ZaloPay
Xác nhận thanh toán
3.2.6. Quản lý đơn hàng
Xem lịch sử đặt tour
Xem trạng thái đơn hàng
Hủy đơn (nếu chưa xử lý)
3.2.7. Tài khoản cá nhân
Đăng ký  đăng nhập
Cập nhật thông tin cá nhân
Đổi mật khẩu
4.  Công nghệ dự kiến sử dụng
Frontend HTML, CSS, JavaScript
Backend PHP (Laravel)  NodeJS
Database MySQL
Thanh toán VNPay API, MoMo API
5.  Hướng phát triển mở rộng
Tích hợp đánh giá tour
Chat hỗ trợ trực tuyến
Gợi ý tour theo hành vi người dùng
Ứng dụng mobile
